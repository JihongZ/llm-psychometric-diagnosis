# PTSD_Forbes2018

Forbes et al. (2018) — PHQ-9 and GAD-7 at two timepoints.

**Source:** https://osf.io/6fk3v/

## Folder Contents

| File | Description |
|---|---|
| `items.csv` | Item metadata: text, scale, cut-off, response range |
| `responses.csv` | Item-level responses (generated by `prepare_responses.R`) |

## How to Generate responses.csv

Run the following to download the raw data and write `responses.csv`:

```r
# From project root
source("Code/prepare_responses.R")
```

## How to Run Diagnosis

From the project root in your terminal:

```bash
./bin/diagnosis Projects/PTSD_Forbes2018
```

Or add `bin/` to your PATH once and run from anywhere:

```bash
export PATH="$PATH:/path/to/LLM_Network_Psychometrics/bin"
diagnosis Projects/PTSD_Forbes2018
```

The agent will:
1. Read `items.csv` and `responses.csv`
2. Generate `PHQ-9_diagnosis.R` and `GAD-7_diagnosis.R`
3. Execute both scripts via `Rscript`
4. Write `diagnosis_report.md` with results and interpretation
